= 저장된 기준을 편집합니다
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
이전에 저장한 검색 조건을 편집할 수 있습니다. 저장된 기준에 새 부품을 추가하거나 기존 저장된 기준에서 부품을 제거할 수 있습니다.



== 시작하기 전에

저장된 기준을 로드하여 편집해야 합니다.



== 단계

. 저장된 기준을 로드하고 저장된 기준 링크를 클릭합니다.
. 검색 기준 구체화 * 링크를 클릭합니다. 저장된 기준에서 기준을 추가하거나 제거할 수 있습니다.
