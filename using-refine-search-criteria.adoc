= 검색 기준 구체화를 사용합니다
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
* 검색 기준 구체화 * 를 사용하면 제품의 호환 구성을 탐색할 수 있습니다. 다른 카테고리에서 부품을 선택함으로써, 제품에 대해 원하는 구성으로 결과를 얻을 수 있는 설정을 좁힐 수 있습니다.


NOTE: 키워드 * Rack * 및 * Blade * 는 * 4.x * 의 구성 요소로만 표시됩니다.
