= 검색 기준을 정의하고 입력합니다
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
구성 요소, 솔루션 또는 범주의 조합을 사용하여 검색 기준을 정의할 수 있습니다. 상호 운용성 매트릭스 툴을 사용하면 구성 요소, 솔루션 또는 범주를 추가하여 검색 기준을 입력할 수 있습니다. 검색 텍스트 상자, 솔루션 탐색기 또는 구성 요소 탐색기를 사용하여 검색 기준을 입력할 수 있습니다.
